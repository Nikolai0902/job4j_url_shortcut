## url_shortcut

REST API использующийся для сокращения ссылок и ведения статистики по ним.

### Используемые технологии:
* Java 17
* Maven 3.8
* Spring 2.7.17
* * Web
* * Data Jpa
* * Security
* * Validation
* * Test
* PostgreSQL 14
* Liquibase 3.10.3
* Lombok 1.18.30
* Checkstyle
